# 🏛️ Museu Virtual Premium

Projeto desenvolvido para apresentar a história, cultura e contribuições dos Povos Originários do Paraná e dos Povos Africanos para a formação da sociedade brasileira.

## 📚 Sobre o Projeto

O Museu Virtual Premium é uma exposição digital interativa que busca valorizar a diversidade cultural brasileira por meio de textos informativos, imagens históricas, galeria interativa e atividades educativas.

## 🌿 Temas Abordados

### Povos Originários do Paraná
- Povos Guarani, Kaingang e Xetá
- Tradição oral
- Cultura e espiritualidade
- Cerâmica indígena
- Artesanato tradicional

### Povos Africanos
- História da diáspora africana
- Resistência cultural
- Influência na música, dança e culinária
- Formação da identidade brasileira
- Contribuições para a cultura do Paraná

## ✨ Funcionalidades

- Página inicial moderna
- Design responsivo
- Galeria de imagens
- Linha do tempo histórica
- Quiz interativo
- Navegação suave
- Layout otimizado para celular e computador

## 🛠️ Tecnologias Utilizadas

## 🚀 Como Executar

1. Faça o download do projeto.
2. Adicione suas imagens na pasta `imagens`.
3. Coloque a imagem principal como `capa.jpg`.
4. Abra o arquivo `index.html` no navegador.

Ou acesse pela publicação no GitHub Pages.

## 🎯 Objetivo Educacional

Este projeto foi desenvolvido com fins educacionais para promover o conhecimento sobre a diversidade cultural brasileira, destacando a importância histórica e cultural dos povos originários e africanos.
